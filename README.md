Python script that connect rss with vk public. I host it on Heroku. MongoDB is also needed.

Publics for example:

* https://vk.com/vk_bashorg
* https://vk.com/vk_ith


Packages using:

 1. [vk_api](https://github.com/python273/vk_api)
 2. APScheduler
 3. feedparser
 4. pymongo
 5. requests
